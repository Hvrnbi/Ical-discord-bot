# ical-discord-bot
___

Ical discord bot is a bot that sends your attends for today from an ical.

___

## Deploy the bot

- Clone this repo

- Create an application on https://discord.com/developers/, go to Bot and click Reset Token

- Put your token at the bottom of the main.py file, where it is written to

- Create a data folder at the root of the project and paste your calendar in

- Rename your calendar "cal.ics"

- Create a venv and run ```pip install -r requirements.txt```

- Run the main file

- Add the bot to your server

That's it !

You can now view your events for today with the ```edt?``` command (by default)
